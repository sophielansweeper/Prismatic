## Description

The **data mapper** component allows you to apply a _map_ to a value or list of values.
This is handy if you have a list of items, and would like to categorize each item.

For example, if you have a list of input that reads:

```json
["apple", "orange", "carrot"]
```

and a map that reads:

```json
{
  "apple": "fruit",
  "blueberry": "fruit",
  "carrot": "vegetable",
  "green bean": "vegetable",
  "orange": "fruit"
}
```

then a **value list mapper** action would output `["fruit", "fruit", "vegetable"]`.
